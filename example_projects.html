Setup Instructions

Click here to see how to download and open the BVSDK project.

Setup

  1. Clone the BVSDK repository: 
    $ git clone https://github.com/bazaarvoice/bv-android-sdk.git
  2. In Android Studio select Open an existing Android Studio Project, or File > Open.
  3. Navigate to the root directory of the project, and click OK
  4. Add your BV generated config files to the demo apps asset directory.
  5. To set the environment (production versus staging configuation), set the public static final BazaarEnvironment ENVIRONMENT value in the DemoConstants.java project file.

TIP: When running the Conversations API demo, you can set your own test product Ids in the DemoConversationsAPIFragment.java file.

BV Demo app

The BV Demo app can be used as an example of best practices to use when integrating the BVSDK into your app mobile application. It features a mock eCommerce application known as Endurance Cycles. Think of this as a Kitchen Sink style app.

  If you are running on an emulator, the examples require that you have Google Play Services installed.


  1. Right click the DemoRecsHomeActivity.java file, and select Run.



  2. On your device, you should see a home page like the one below. This app can be used as an example of best practices to use when integrating the BVSDK into your app mobile application.

  3. To see your own data in this app, tap the settings cog to go to the settings page, then tap "Select Preloaded Profile". You will be presented with the choice to change the data source between "Mock Endurance Cycles" (which is a Bazaarvoice demo source), and your BV generated config. The BV generated config will automatically display the environment set in the DemoConstants.java file.

BV API app

The BV API app contains simple minimal code examples for each of the products and how to set up the API calls for specific use cases.

  If you are running on an emulator, the examples require that you have Google Play Services installed.

  1. Right click the DemoMainActivity.java file, and select Run.



  2. On your device, you should see a home page like this. This app can be used to view example implementations of the different parts of the BVSDK.



  3. There will be a drawer with simple code example for each of the products.

  4. To see your own data in this app, tap the settings cog to go to the settings page, then tap "Select Preloaded Profile". You will be presented with the choice to chance the data source between "Mock Endurance Cycles" (which is a Bazaarvoice demo source), and your BV generated config. The BV generated config will automatically display the environment chosen in the DemoConstants.java file.
