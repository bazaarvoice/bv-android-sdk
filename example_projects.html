Example Projects

The BVSDK contains basic example projects for each module as well as a full demo project highlighting a full integration of all the BVSDK components. Preview any of the samples in the SDK to help you get your implementation up and running quickly.

  If you are running on an emulator, the examples require that you have Google Play Services installed.


Configuring the Code Example App

Click here to see how to configure the code example app.

  1. Open the DemoConstants.java file, and modify...

    1. BV_CLIENT_ID - your Bazaarvoice provided clientId string name
    2. PASSKEY_SHOPPER_AD - your Bazaarvoice Shopper Advertising api key
    3. PASSKEY_CONVERSATIONS - your Bazaarvoice Conversations api key
    4. PASSKEY_CURATIONS - your Bazaarvoice Curations api key
    5. PASSKEY_LOCATION - your Bazaarvoice Location api key
    6. ENVIRONMENT - the environment you want from the BazaarEnvironment enum

    BV_CLIENT_ID and ENVIRONMENT are required, and the others may be left as "REPLACE_ME" if not used



  2. Right click the DemoMainActivity.java file, and select Run.



  3. On your device, you should see a home page like this. This app can be used to view example implementations of the different parts of the BVSDK.



  4. There will be a drawer with simple code example for each of the products.

Configuring the Demo App

Click here to see how to configure the demo app.

  1. Open the DemoConstants.java file, and modify...

    1. BV_CLIENT_ID - your Bazaarvoice provided clientId string name
    2. PASSKEY_SHOPPER_AD - your Bazaarvoice Shopper Advertising api key
    3. PASSKEY_CONVERSATIONS - your Bazaarvoice Conversations api key
    4. PASSKEY_CONVERSATIONS_STORES - your Bazaarvoice Conversations api key
    5. PASSKEY_CURATIONS - your Bazaarvoice Curations api key
    6. ENVIRONMENT - the environment you want from the BazaarEnvironment enum

    BV_CLIENT_ID and ENVIRONMENT are required, and the others may be left as "REPLACE_ME" if not used.

      The demo app is not useable without a Shopper Advertising API Key.



  2. Right click the DemoRecsHomeActivity.java file, and select Run.



  3. On your device, you should see a home page like this. This app can be used as an example of best practices to use when integrating all parts of the BVSDK into a single app.
