Example Projects

Product Recommendations, Advertising, Curations, and Conversations are all demonstrated in an example app, which shows how to properly implement these modules. Once you have cloned this repository, you can configure and run the example app.

  If you are running on an emulator, the examples require that Google Play Services is installed.


Configuring the Examples

  1. Open the example app by selecting File > Open and choose the repository root.


  2. Open up UserConfigurationImpl.java and add your SDK configuration values in the REPLACE_ME text. Add your client ID and the API keys for your desired modules here.

  3. See the value for BV_ENVIRONMENT in UserConfigurationImpl.java and set the evironment for staging or production.

  4. You should now be ready to run!
